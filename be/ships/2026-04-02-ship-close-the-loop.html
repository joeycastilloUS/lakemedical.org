Auth Flow

Browser:
  1. user types username + TOTP code
  2. auth-gate.js calls nous_auth(user, totp)
  3. nous.js builds wire triples, calls wire_pack()
  4. POST /api/wire with binary NTRP frame

Worker (Cloudflare):
  5. Validates NTRP0001 magic
  6. Derives cockpit key: HMAC-SHA-256(PSK, "nous-transport:cockpit")
  7. AES-256-GCM encrypt into 0xCA envelope
  8. TCP connect relay.3-nous.net:8080, send, shutdown(WR), read

Relay:
  9. Detects 0xCA magic, decrypts with cockpit key
  10. Parses NTRP: extracts action=auth, user, totp
  11. find_first_prime() -> us-west2 prime
  12. Re-encrypts with transport key: HMAC-SHA-256(PSK, "nous-transport:us-west2")
  13. TCP to prime, send, read response

Prime:
  14. Detects 0xCA, decrypts with transport key
  15. wire_handle_auth(): looks up user, validates TOTP
  16. Returns (resp, status, ok) or (resp, status, rejected)

Relay (return):
  17. Decrypts prime response
  18. If ok: mints session (64-hex token, 900s TTL, IP-bound)
  19. Returns: token + ttl + status=ok as wire triples
  20. Re-encrypts with cockpit key for Worker

Worker (return):
  21. Decrypts 0xCA response
  22. Returns plaintext NTRP to browser

Browser (return):
  23. wire_unpack() response triples
  24. nous_session_save(token) to sessionStorage
  25. auth-gate dismisses, protected content visible
  26. Session timer starts (15 min countdown)
