31.1 — Replace qr.js with faithful C port (5 pts)

Port every function from C:\metal\auth\qr.c (606 lines) to JavaScript. Includes:

  • GF(256) log/exp tables + multiply
  • Reed-Solomon generator + encode
  • Bit buffer for byte-mode encoding
  • Version/capacity tables (matching C exactly)
  • BCH(15,5) format info with mask 0x5412
  • Matrix placement: finders, timing, alignment, format, data
  • All 8 mask patterns + all 4 penalty rules
  • Multi-block interleaving
  • Same API: qr_encode(text) → {data, size}

5 pts — substantial but mechanical. The C source is the spec. No design decisions.
