alumni/js/nous.js — High-level NOUS wire client for the browser

  • Wraps wire.js pack/unpack with fetch('/api/wire')
  • Binary transport: Content-Type: application/octet-stream
  • Auth triples match relay_server.c handle_wire_auth() exactly

  • Session stored in sessionStorage['nous-session']

API Surface

  • nous_send(triples) — Core: pack → POST /api/wire → unpack response
  • nous_auth(user, totp) — Login: {auth,action} + {user,entity} + {totp,signal}
  • nous_register(user) — Register: {register,action} + {user,entity}
  • nous_request(token, triples) — Authenticated: prepends {token,meta,session}
  • nous_session_check() — Validates stored session against relay
  • nous_session_save(token) / get() / clear() — sessionStorage wrapper
  • nous_find(triples, predicate) — Extract subject by predicate
  • nous_status(triples) — Extract {resp,status,<value>} object field
