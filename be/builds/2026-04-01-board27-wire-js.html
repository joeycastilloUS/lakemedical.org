What Was Added

alumni/js/wire.js — Exact JS port of kASTIL/nous/comms/skill/wire/wire.c

  • Binary triple format: {subject, predicate, object}
  • Frame layout: [8B magic "NTRP0001"][4B count LE][4B payload_bytes LE][payload]
  • Each triple: [2B subject_len LE][subject][2B predicate_len LE][predicate][2B object_len LE][object]
  • Byte order: LITTLE-ENDIAN — matches wire.c w16()/w32()/r16()/r32()
  • Max frame: 128KB (WIRE_BUF_SIZE)

API Surface

  • wire_pack(triples) — Array of {s, p, o} → Uint8Array NTRP frame
  • wire_unpack(buf) — Uint8Array → Array of {s, p, o}
  • wire_is_ntrp(buf) — Check magic header
  • wire_find(triples, predicate) — First match by predicate, returns subject
  • wire_find2(triples, subject, predicate) — Match by subject+predicate, returns object
  • wire_action(triples) — Shorthand for wire_find(triples, 'action')
  • wire_entities(triples) — All triples with predicate='entity'
