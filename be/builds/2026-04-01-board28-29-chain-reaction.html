Board 29.1 — wire.js Worker with 0xCA Envelope (5 pts) ✔

  • Cloudflare Worker proxy: POST /api/wire
  • Flow: validate NTRP0001 magic → derive cockpit key (HMAC-SHA-256) → encrypt 0xCA envelope (AES-256-GCM) → TCP connect relay.3-nous.net:8080 → decrypt response → return NTRP
  • Envelope: [0xCA][nonce:12B][ciphertext+tag] — 29 byte overhead
  • Rate limiting: 10 req/min per IP, returns 429
  • ZERO hardcoded IPs — DNS resolves at runtime
  • File: functions/api/wire.js (166 lines)

Decisions Made

  • QR from scratch: Zero dependencies. ISO 18004 compliance. No CDN, no library, no external calls.
  • Cockpit key, not region key: Worker derives nous-transport:cockpit — relay tries cockpit first.
  • WebCrypto [ct+tag] layout: Byte-compatible with C's separate [ct][tag] — no translation needed.
  • Session in sessionStorage: Tab-scoped, clears on close. No localStorage persistence.
  • Rate limiting in-isolate: Simple, resets per-isolate. Good enough for relay proxy protection.
